Welcome to my website!

My name is James Dean; my career as a scientist has focused heavily on solving chemical problems using cutting edge computational tools. I leverage machine learning or run computational chemistry calculations to solve challenging problems, and then leverage my software engineering experience to deploy the solution to users.

This website has some more details about who I am and what I've worked on in the past.

Some more detail on what motivated me to make this site can be found on this page, but the long and short is that this site was not made using any framework! While the modern webdev toolkit can do wonders (GitHub Pages will let you run Jekyll), and even though I am experienced in building sites using modern tools like Flask, Redux, and React, I wanted the challenge of learning to make a site from scratch using only HTML, CSS, and JS. It's a throwback to an era where the web was handcrafted and more personal.
